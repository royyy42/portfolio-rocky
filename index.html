Proyek Utama

Sistem Kasir Restoran Berbasis Web

Proyek akademik / tugas kuliah
PHP MySQL JavaScript Bootstrap

Proyek ini merupakan sistem kasir berbasis web yang dibuat untuk membantu proses pemesanan, transaksi, dan pengelolaan data pada restoran. Sistem dikembangkan agar pengolahan data menjadi lebih rapi, cepat, dan terstruktur dibandingkan pencatatan manual.

Tujuan Proyek

Membuat sistem kasir berbasis web yang dapat membantu proses transaksi restoran, menghitung total pembayaran, mencetak struk, dan menyajikan laporan transaksi.

Masalah yang Diselesaikan

Pencatatan manual memperlambat pelayanan, meningkatkan risiko kesalahan perhitungan, dan menyulitkan pengelolaan data transaksi.

Fitur Utama

  • Login admin
  • Dashboard / halaman utama
  • Pengelolaan data pesanan
  • Pengelolaan transaksi
  • Perhitungan total pembayaran
  • Cetak struk
  • Laporan transaksi
  • Tambah, edit, dan hapus data
